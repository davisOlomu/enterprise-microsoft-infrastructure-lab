# What I've Learned So Far

Working through this lab has helped me move beyond simply configuring Microsoft technologies to understanding **why** organisations implement them.

Some of the questions I've explored include:

- Why do organisations use Active Directory instead of local accounts?
- Why are security groups preferred over assigning permissions directly to users?
- Why are mapped drives deployed using Group Policy?
- How do NTFS permissions and security groups work together?
- Why is DNS fundamental to Active Directory?
- How do governance, security, and auditing complement one another?
- How does hybrid identity simplify enterprise identity management?

Exploring these questions has helped me better understand how Microsoft technologies are used in real-world enterprise environments.
